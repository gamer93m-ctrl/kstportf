# Константин — Product Designer (Senior UX/UI)

Портфолио продуктового дизайнера: кейсы АМА, ИТ Контакт, Росбанк, Здоровье.

## Стек

- React 19 + TypeScript
- Vite
- Tailwind CSS v4
- Framer Motion

## Особенности

- Фиксированная левая панель с фото и описанием, скроллится только правая колонка с кейсами
- Плавающая нижняя навигация со scroll-spy и анимированной активной «пилюлей»
- Сворачиваемые подсекции кейсов, плавные появления блоков при скролле
- Адаптивная вёрстка (на мобильных панель становится шапкой)

## Запуск

```bash
npm install
npm run dev      # dev-сервер на http://localhost:5173
npm run build    # прод-сборка в dist/
```
